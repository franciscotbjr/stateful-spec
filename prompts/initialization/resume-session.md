# Prompt: Resume Previous Session

## Context

Use this prompt when returning to a project after a break. It restores the AI's understanding of where you left off, what decisions were made, and what needs to happen next.

## Prerequisites

- You have a **Project Profile** for this project
- You have a **session summary** from the previous session (generated by `prompts/operations/save-session.md`)

## Input

Paste the following alongside this prompt:

1. Your **Project Profile**
2. The **session summary** from your last session
3. Optionally: any new context since the last session (new requirements, resolved blockers, etc.)

## Prompt

<!-- Copy from here -->

You are an AI development assistant resuming work on an ongoing project. I will provide you with the Project Profile and a session summary from our last working session. Please review both and pick up where we left off.

**Your role:**
- Follow the Design Source methodology: Analyze → Plan → Specify → Implement → Verify
- Respect all conventions and constraints defined in the Project Profile
- Continue from the exact point described in the session summary
- Do not redo work that was already completed

**Here is the Project Profile:**

{{PROJECT_PROFILE}}

**Here is the session summary from last time:**

{{SESSION_SUMMARY}}

**New context since last session (if any):**

{{NEW_CONTEXT_OR_NONE}}

Please:
1. Confirm you understand the current state of the project
2. Summarize where we left off (current phase, in-progress work, pending decisions)
3. List the immediate next steps
4. Highlight any blockers or open questions that need resolution before proceeding

<!-- To here -->

## Expected Output

The AI should respond with:
1. Confirmation of the project context
2. Summary of the previous session's state
3. Clear next steps, ordered by priority
4. Any blockers or questions

## Next Steps

- Resolve any blockers or answer questions
- Use the appropriate phase transition prompt to continue work
- If the AI is ready to continue implementation, proceed directly
